# ベイの操作アルゴリズム

---

# 1. ベイの基本物理モデル

ベイは以下の状態を持ちます。

```
position (x, y)
velocity (vx, vy)
rotationSpeed
energy
mass
```

意味

| パラメータ | 意味 |
| --- | --- |
| position | 位置 |
| velocity | 移動速度 |
| rotationSpeed | 回転力 |
| energy | スタミナ |
| mass | 重さ |

---

# 2. スマホ入力

スマホから送る値

```
tiltX
tiltY
shakePower
```

意味

| 値 | 用途 |
| --- | --- |
| tiltX | 左右操作 |
| tiltY | 前後操作 |
| shakePower | 発射強度 |

---

# 3. 発射アルゴリズム

スマホを振ると発射。

発射速度

```
launchPower = clamp(shakePower, 0, MAX_POWER)
```

初速

```
vx = randomDirection * launchPower
vy = randomDirection * launchPower
```

ベイらしさを出すため

**少しランダム性**

を入れる。

---

# 4. 移動制御

スマホ傾きを **加速度**として扱う。

```
ax = tiltX * controlForce
ay = tiltY * controlForce
```

速度更新

```
vx += ax * dt
vy += ay * dt
```

位置更新

```
x += vx * dt
y += vy * dt
```

---

# 5. 摩擦（重要）

ベイは徐々に減速。

```
vx *= friction
vy *= friction
```

例

```
friction = 0.98
```

これで

**回転体の滑り感**が出る。

---

# 6. 回転エネルギー

ベイは回転しているので

**時間で弱くなる**

```
rotationSpeed -= spinDecay * dt
```

エネルギー

```
energy = rotationSpeed
```

---

# 7. 回転による安定性

回転が高いほど

**操作が効きにくい**

これがベイっぽい。

```
controlFactor = 1 / (1 + rotationSpeed * 0.1)
```

入力

```
ax *= controlFactor
ay *= controlFactor
```

---

# 8. 壁衝突

アリーナ壁

```
if hitWall:
    vx *= -0.8
    vy *= -0.8
```

少し減衰

---

# 9. ベイ衝突

2つのベイが衝突

```
impact = relativeVelocity * mass
```

エネルギー減少

```
energy -= impact * 0.1
```

ノックバック

```
vx += impactX
vy += impactY
```

---

# 10. 弾き飛ばし

衝突時

```
knockback = impact * knockbackFactor
```

適用

```
vx += knockbackX
vy += knockbackY
```

---

# 11. ノックアウト判定

アリーナ外

```
distanceFromCenter > arenaRadius
```

↓

敗北

---

# 12. スタミナ敗北

```
if energy <= 0:
    eliminated
```

# 13. 操作感チューニング

重要パラメータ

| パラメータ | 効果 |
| --- | --- |
| controlForce | 操作の効き |
| friction | 滑り具合 |
| spinDecay | スタミナ減少 |
| knockbackFactor | 衝突の激しさ |

---

# 14. ベイっぽさを出す小技

### 回転ドリフト

少し横滑り

```
drift = perpendicular(velocity) * spinFactor
```

---

### ランダム揺れ

回転が弱いと

```
vx += randomNoise
vy += randomNoise
```

# 15. サーバゲームループ

```
Game Tick

1 入力取得
2 加速度計算
3 速度更新
4 摩擦適用
5 位置更新
6 衝突判定
7 エネルギー減少
8 状態配信
```

---

# 16. Tick疑似コード

```jsx
function gameTick(dt) {

  players.forEach(p => {
    bey = p.bey

    ax = p.tiltX * controlForce
    ay = p.tiltY * controlForce

    bey.vx += ax * dt
    bey.vy += ay * dt

    bey.vx *= friction
    bey.vy *= friction

    bey.x += bey.vx * dt
    bey.y += bey.vy * dt
  })

  resolveCollisions()
}
```

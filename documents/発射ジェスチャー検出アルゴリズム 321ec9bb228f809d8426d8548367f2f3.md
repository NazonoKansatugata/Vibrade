# 発射ジェスチャー検出アルゴリズム

スマホの「振る」をそのまま加速度のしきい値で判定すると、**誤作動が多い**（持ち替え・操作時の揺れなど）。

そこで **ジェスチャーとしての“発射動作”を段階検出**するアルゴリズムにします。

---

# 1. 発射ジェスチャーの基本イメージ

ベイブレードの発射動作は

```
引く → 強く振る → 離す
```

スマホで表現すると

```
静止
↓
加速
↓
ピーク
↓
減速
```

つまり **加速度の波形パターン**を検出します。

---

# 2. 使用センサー

ブラウザAPI

```jsx
DeviceMotionEvent
```

取得値

```
accelerationIncludingGravity.x
accelerationIncludingGravity.y
accelerationIncludingGravity.z
```

---

# 3. 加速度の合成

3軸をまとめる

```
accMagnitude = sqrt(ax² + ay² + az²)
```

JS例

```jsx
const magnitude = Math.sqrt(ax*ax + ay*ay + az*az)
```

---

# 4. ノイズ除去（重要）

センサーはかなりノイズがあります。

**移動平均フィルタ**

```
filtered = (prev * 0.8) + (current * 0.2)
```

JS

```jsx
filteredAcc = prevAcc * 0.8 + magnitude * 0.2
```

---

# 5. ジェスチャー検出ステートマシン

発射は **状態遷移**で検出します。

状態

```
IDLE
PULLING
LAUNCH
COOLDOWN
```

---

# 6. 状態1：IDLE

スマホが静止している状態

条件

```
accMagnitude < 12
```

ここから

```
accMagnitude > pullThreshold
```

で

**PULLING**

---

# 7. 状態2：PULLING

スマホを振り始めた状態

条件

```
accMagnitude > pullThreshold
```

例

```
pullThreshold = 14
```

ピーク検出

```
accMagnitude > launchThreshold
```

↓

**LAUNCH**

---

# 8. 状態3：LAUNCH

発射確定

```
launchThreshold = 20
```

この瞬間

```
emit("launchBey")
```

---

# 9. 状態4：COOLDOWN

連続発射防止

```
cooldown = 1000ms
```

1秒間は発射不可

↓

終了後

```
IDLE
```

---

# 10. 発射強度

ピーク加速度で決める

```
power = clamp(accMagnitude / 30, 0, 1)
```

例

```
20 → power 0.6
30 → power 1.0
```

---

# 11. 完成アルゴリズム

```
IDLE
 ↓ acc > pullThreshold

PULLING
 ↓ acc > launchThreshold

LAUNCH
 ↓ emit launch event

COOLDOWN
 ↓ 1s

IDLE
```

---

# 12. 擬似コード

```jsx
let state = "IDLE"
let lastLaunch = 0

function detectLaunch(acc) {

  const now = Date.now()

  if(state === "COOLDOWN"){
    if(now - lastLaunch > 1000){
      state = "IDLE"
    }
    return
  }

  if(state === "IDLE"){
    if(acc > 14){
      state = "PULLING"
    }
  }

  if(state === "PULLING"){
    if(acc > 20){
      launch(acc)
      lastLaunch = now
      state = "COOLDOWN"
    }
  }
}
```

# 13. 誤検知対策

重要ポイント

### ① 最小時間

```
PULLING 継続 > 50ms
```

---

### ② 方向判定

前方向のみ

```
accZ > 0
```

---

### ③ 回転無視

傾き操作は

```
orientation
```

発射は

```
acceleration
```

で分離
